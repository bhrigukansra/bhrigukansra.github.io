Shipping high-quality features is mostly about reducing uncertainty early.

## Workflow

- Define the user problem in one sentence.
- Build the smallest useful prototype.
- Validate quickly with peers or users.
- Iterate with analytics and feedback.

Fast loops beat perfect plans.
